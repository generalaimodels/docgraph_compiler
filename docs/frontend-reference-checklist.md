# Frontend Reference Checklist

- Reader is a strict docs shell: top docs bar, left navigation tree, center article, right TOC/context rail.
- Conversion contains import workflows and job status only.
- Metrics contains evaluation, fidelity, diagnostics, and platform health only.
- Reader contains no dashboard hero blocks, cross-page launch cards, or mixed product controls.
- Visual polish happens only after the structural split above is satisfied on desktop and mobile.
